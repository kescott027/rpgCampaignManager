## üß© ENTITY

> Tracks: People, Places, Groups, and Things in the campaign world.

### ENTITY

- **Entity ID**: *String* ‚Äî Unique canonical name of the entity.
- **Entity Type**: *Enum* ‚Äî One of:
  - `GM`
  - `Player`
  - `Player Character`
  - `Non Player Character`
  - `Group`
  - `Place`
  - `Thing`
- **Entity Aliases**: *List[String]* ‚Äî Alternate names or nicknames.
- **Entity Status**: *List[String]* ‚Äî Describes the current or past state(s) of the entity.
- **Entity Relationships**: *List[Relationship]* ‚Äî Key links to other entities.

## üîó RELATIONSHIP

> Tracks: Multi-entity affiliations, alliances, rivalries, and bonds.

### RELATIONSHIP

- **Relationship ID**: *String* ‚Äî Unique 20-character random string.
- **Relationship Principal**: *String* ‚Äî The `Entity ID` this relationship centers on.
- **Relationship Description**: *String (‚â§100 characters)* ‚Äî Describes the relationship.
- **Relationship Related Entities**: *List[String]* ‚Äî Other `Entity ID`s involved.
- **Relationship Tags**: *List[String]* ‚Äî Includes:
  - Principal
  - Related entities
  - Extracted verbs (e.g. ‚Äúfled‚Äù, ‚Äúcommands‚Äù)
  - Standard tag phrases (e.g. `[Fleeing]`, `[Allied with]`)

## üìÜ EVENT

> Tracks: Time-stamped, narratively or mechanically important actions.

### EVENT

- **Event Timestamp**: *String* ‚Äî Auto-generated, format: `yyyyMMdd:HH:mm:ss.SSS`
- **Event Description**: *String (‚â§100 characters)* ‚Äî Must be in past tense and use `[Entity]` references.
- **Source**: *Enum* ‚Äî One of `[Campaign]`, `[Novel]`, `[Both]` (default: `[Campaign]`)
- **Book**: *Integer* ‚Äî Book number (default: 0)
- **Chapter**: *String* ‚Äî Chapter name (default: `'Unspecified'`)
- **Session**: *String* ‚Äî Campaign session label
- **Index**: *Integer* ‚Äî Default 1, auto-incremented to avoid duplicate Event IDs
- **Event ID**: *String* ‚Äî Generated as:  
  `{Source}:{Book}:{Chapter}:{Session}:{Timestamp}:{Index}`

## üß† STATUS

> Tracks: The condition of an entity and its evolution through events.

### STATUS

- **Status Event ID**: *String* ‚Äî Links to triggering `Event ID`
- **Status Principal**: *String* ‚Äî The `Entity ID` being described
- **Status ID**: *String* ‚Äî Generated as: `{Status Principal}:{Status Event ID}`
- **Status Related Entities**: *List[String]* ‚Äî Other `Entity ID`s involved
- **Status Description**: *String (‚â§100 characters)* ‚Äî Uses past tense and `[Entity]` format
- **Status Active**: *Boolean* ‚Äî True if this is the entity's current condition
- **Status Tags**: *List[String]* ‚Äî Includes:
  - Principal (index 0)
  - Related Entities
  - Key verbs from description
  - Standard tag phrases (e.g., `[Residing in]`, `[Traveling to]`)
